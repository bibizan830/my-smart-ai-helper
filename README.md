# my-smart-ai-helper
## описание проекта
мой проект - сайт с мозгами на python сайт умеет рекомендовать фильмы по ввоу настроение пользователя
## стек
я использую HTML CSS python для разроботки данного сервиса  релизован фулстак сайт с созданием локального сервера и подключением нейросети к сайту использованый библеотеки flask pipeline nf_xet
модель нейросети для генераций идей фильмов - sberbank-ai нейросети для понимания и анализа настроения -blanchefort
## устоновка бибилеотек 
для устоновки библеотек вам необходимо ввести следующую строчку кода:
```
pip install -r requirements.txt
```'
или устоновить библиотеку по одтельности:
```
pip install flask
pip install transformers
pip install pipeline
pip install torch
```
